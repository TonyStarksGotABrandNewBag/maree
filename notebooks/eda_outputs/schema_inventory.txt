=== Notes ===
- The dataset has NO 'Label' column. Class is encoded by file location:
  malware samples live in malware-by-day/YYYY-MM-DD.csv,
  goodware samples live in goodware.csv.
- The Quantic PDF specifies 27 input attributes + a 'Label' column.
  The Brazilian dataset has more raw columns. We will select the
  numeric subset, drop hash/identifier columns (MD5, SHA1, Name),
  optionally feature-engineer from string columns (ImportedDlls
  cardinality, etc.), and construct the Label column from file source.
